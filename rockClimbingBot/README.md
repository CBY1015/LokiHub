### Bot介紹 ###
運動攀登，俗稱攀岩，近年來獲得越來越多的關注，亦於2021年東京奧運，首度列為正式運動項目。伴隨著這股熱潮，此Bot應運而生，其提供了想在台灣攀岩的新手會遇到的，各種疑難雜症的解答，舉凡攀岩需要的裝備、治裝花費、攀岩規則、岩點介紹等，都可詢問，即便是老手，也可從對話中快速取得各縣市岩館的資訊。歡迎對攀岩有興趣的朋友下載使用！
