#Project: FinBot 金融應用機器人

##安裝方法

1. 登入 [Loki](https://api.droidtown.co/loki/) 網頁
2. 新建一個專案`FinBot`
3. 選擇`ArticutModel`並選取`ref/Exchange.ref`後點擊`讀取意圖`按鈕
4. 進入`Exchange` 點擊`生成模型 (Exchange)`按鈕
5. 回到專案頁面複製`FinBot`金鑰
6. 編輯 `FinExchange.py` 填入
	- USERNAME 註冊的帳號 (信箱)
	- API_KEY `Articut`金鑰
	- LOKI_KEY `FinBot`金鑰
7. 開始使用 FinBot
